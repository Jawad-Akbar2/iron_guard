# Static Assets

Place the following files in this directory:

## Images
- `favicon.ico` - Browser tab icon
- `logo.png` - App logo (100x100px)
- `logo-dark.png` - Dark theme logo

## Icons (Optional - Using Lucide React instead)
All icons are handled via lucide-react package

## CSS
- Global styles are in src/index.css
- Tailwind CSS handles all styling

No additional static assets are required for core functionality.
